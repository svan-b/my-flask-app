About the STEMCELL Project

The STEMCELL Project offers a comprehensive financial model that includes an operating model, updated public comparables, and deal comps. Key features of the model include:

  • An operating model derived from actual financials for 2020 and 2021, along with projections from 2022 onwards.
  • Public comparables based on a carefully selected comp group, updated with the latest close of publicly traded companies. The public comps utilize data from 2019 to 2021 (sourced from SEDAR, EDGAR, etc.) and current market data to derive a real-time enterprise value.
  • Deal comps constructed from available data in company reports, press releases, and financial statements.
  • Application of multiples from public comps to STEMCELL's projected values to achieve valuations.
  • A Discounted Cash Flow (DCF) analysis feeding from the operating model, utilizing STEMCELL's weighted average cost of capital based on Thermo Fisherâ€™s WACC calculation.
  • Flexibility to adjust and update the model based on internal estimates and changes in financial inputs.

This dynamic and interactive tool is designed to provide an in-depth analysis of STEMCELL's financial standing, offering valuable insights for strategic decision-making.
